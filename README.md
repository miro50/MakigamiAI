# 🚀 Makigami AI - Lean Process Mapping Engine

**Makigami AI** è un'applicazione web avanzata che trasforma descrizioni in linguaggio naturale (testo o audio) in diagrammi Makigami interattivi e strutturati. 

Pensato per consulenti Lean Six Sigma e Master Black Belt, il tool automatizza il processo di mappatura dello stato "As-Is", calcola istantaneamente l'efficienza del processo (PCE) e genera raccomandazioni "To-Be" basate sul framework ECRS, facendo risparmiare ore di data-entry e disegno manuale.
